# E-commerce Funnel & Customer Satisfaction Analysis

## 📌 Overview
This project analyzes the e-commerce order funnel to identify drop-offs and evaluate how delivery performance impacts customer satisfaction. It combines data processing using Python and visualization using Power BI.

---

## 🎯 Objectives
- Analyze order funnel stages and identify drop-offs
- Measure delivery performance using key KPIs
- Evaluate the impact of late deliveries on customer satisfaction
- Build an interactive dashboard for business insights

---

## 🛠️ Tools & Technologies
- Python (Pandas)
- Power BI
- DAX
- Google Colab

---

## 🔄 Workflow
1. Data loading and cleaning using Python
2. Feature engineering (delivery_time, is_late)
3. Data merging and analysis
4. Visualization using Power BI dashboard

---

## 📊 Key Metrics
- Average Delivery Time
- Percentage of Late Deliveries
- Average Customer Rating
- Order Funnel Drop-offs

---

## 🔍 Key Insights
- Late deliveries significantly reduce customer satisfaction
- Delivery time is a key driver of customer ratings
- Funnel analysis reveals drop-offs at delivery and cancellation stages

---

## 📁 Project Structure
- notebooks/ → Python analysis (Colab)
- dashboard/ → Power BI dashboard
- data/ → raw datasets
- outputs/ → processed dataset

---

## 📌 Conclusion
This project demonstrates how delivery performance directly impacts customer experience and highlights operational inefficiencies using data-driven insights.
